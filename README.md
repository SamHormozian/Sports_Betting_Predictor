# Sports Betting Predictor

## Overview
This project aims to predict sports outcomes (e.g., match winners, point spreads, or over/under scores) using a neural network built from scratch. It focuses on data from major sports leagues and utilizes historical data, team/player statistics, and betting odds.

## Features
- Predict match winners for NBA, NFL, soccer, and more.
- Analyze and visualize trends in sports data.
- Simulate betting strategies and calculate ROI.

## Repository Structure
- `data/`: Raw and processed data files.
- `notebooks/`: Prototyping and exploratory data analysis.
- `src/`: Source code for the project.
- `visualizations/`: Charts and graphs for evaluation.

## How to Run
1. Clone the repository:
    ```
    git clone https://github.com/SamHormozian/Sports_Betting_Predictor.git
2. Install Dependencies:
    pip install -r requirements.txt
3. Run training pipeline:
    python src/train.py
    ```
